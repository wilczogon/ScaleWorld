# ScaleWorld
Just a simple node.js project

# How to start
First download MySQL database from official site.
Add mysql home directory to PATH. Then:
> mysql -u user_name -p < db_setup.sql
> mysql -u user_name -p < example_data.sql

Install Node.js.
Run:
> npm install express
> npm install mustache-express
> npm install mysql

Then start your application:
> node app.js

Now, open in your browser following url:
127.0.0.1:8081
